# e2e/specs/

Browser flows live here as `*.flow.json` (the runner in `run.ts` picks up
**only** that extension — see `../README.md` before touching them).

Also home to **single-module SDD feature specs** (`SPEC-NN-*.md`) written by
the `spec-creator` skill; the runner ignores `.md` files, so they coexist
safely. Cross-module specs live in repo-root `specs/`.
